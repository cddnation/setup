CDD Development Setup - MAC OS X
=====

Get setup for developing on the CDD stack (OSX). Run the commands below to get setup in a jiffy.

__FED Stack__

Will give you the following:

* Node and NPM
* Yeoman
* Grunt
* Bower
* Sublime text
* GIT

__BE Stack__

Will give you the following:

* Composer
* MAMP Pro
* Sublime text
* GIT
